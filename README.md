## 🚀 快速开始 / Quick Start

### 环境要求

- Node.js >= 14.0.0

### 安装与运行

```bash
# 克隆仓库
git clone https://github.com/MasterPick/md-preview-tool.git
cd md-preview-tool

# 直接启动（无需 npm install，无外部依赖）
node server.js

# 或使用 npm
npm start
```

启动后，打开浏览器访问：**http://localhost:3000**

## 📖 使用说明 / Usage

1. **编辑**：在左侧编辑器中输入 Markdown 内容
2. **预览**：右侧实时显示渲染后的 HTML 效果
